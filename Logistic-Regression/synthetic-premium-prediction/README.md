# Synthetic Premium Prediction - README

## Overview

The **Synthetic Premium Prediction** project is a Machine Learning classification project that predicts whether a customer is likely to purchase a premium subscription/product based on user behavior and demographic data.

This project demonstrates the complete Data Science workflow including:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Feature Encoding
* Data Visualization
* Model Building using Logistic Regression
* Model Evaluation

---

## Features

* Data preprocessing and cleaning
* Handling categorical variables with encoding
* Exploratory Data Analysis using Seaborn and Matplotlib
* Logistic Regression model training
* Accuracy score and classification report evaluation
* Correlation analysis and feature insights

---

## Dataset Information

The dataset contains customer-related information such as:

* Age
* Monthly Income
* Website Visits
* Time Spent on Platform
* Previous Purchases
* Student Status
* Country
* Device Type
* Purchased (Target Variable)

The target variable predicts whether a user purchases the premium product.
