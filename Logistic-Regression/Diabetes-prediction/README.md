# Diabetes Prediction - README

## Overview

The **Diabetes Prediction** project is a Machine Learning classification project designed to predict whether a patient is diabetic based on medical attributes.

This project follows a complete Data Science pipeline from data preprocessing to model evaluation using Logistic Regression.

---

## Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Medical data visualization
* Logistic Regression model training
* Model evaluation and prediction analysis
* Statistical insights from healthcare data

---

## Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

---

## Dataset Information

The dataset includes several medical parameters such as:

* Pregnancies
* Glucose Level
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age
* Outcome (Target Variable)

The target variable determines whether the patient has diabetes.
