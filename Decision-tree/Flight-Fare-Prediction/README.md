# ✈️ Flight Price Prediction using Machine Learning

## 📌 Overview

This project predicts flight ticket prices using Machine Learning. The dataset is cleaned, preprocessed, analyzed, and used to train a Decision Tree Regressor model for airfare prediction.

## 🎯 Objectives

- Understand flight pricing factors
- Perform data cleaning and preprocessing
- Conduct Exploratory Data Analysis (EDA)
- Build a Machine Learning model
- Predict flight ticket prices accurately

## 📂 Dataset Features

- Airline
- Source
- Destination
- Date of Journey
- Duration
- Total Stops
- Additional Information
- Price (Target Variable)

## 🔧 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Exploratory Data Analysis

- Price Distribution Analysis
- Airline-wise Fare Analysis
- Source & Destination Analysis
- Journey Duration Analysis
- Stops vs Flight Price

## 🤖 Model

**Decision Tree Regressor**

The model is trained on preprocessed flight data to predict ticket prices.

## 📈 Evaluation Metrics

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

## 🚀 Project Workflow

1. Data Loading
2. Data Cleaning
3. Feature Engineering
4. Exploratory Data Analysis
5. Feature Selection
6. Model Training
7. Model Evaluation
8. Flight Price Prediction

## 📌 Results

The Decision Tree Regressor successfully learned pricing patterns from historical flight data and generated flight fare predictions.
