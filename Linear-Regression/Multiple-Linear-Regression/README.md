# Multiple Linear Regression 

A Jupyter Notebook exploring Multiple Linear Regression (MLR) using Python.

---

## About

This notebook walks through the end-to-end process of building a Multiple Linear Regression model — from loading and exploring the data to training, evaluating, and interpreting the results.

---

## Notebook Contents

1. **Importing Libraries** — NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn
2. **Loading the Dataset** — Reading and previewing the data
3. **Exploratory Data Analysis (EDA)** — Summary stats, correlation heatmap, pair plots
4. **Data Preprocessing** — Handling missing values, encoding, feature scaling
5. **Model Building** — Fitting the MLR model using Scikit-learn
6. **Model Evaluation** — R², MAE, MSE, RMSE
7. **Visualizations** — Actual vs Predicted plot, Residual analysis
8. **Conclusion** — Summary of findings, model performance interpretation, key takeaways, and suggestions for improvement

---


## Requirements

- Python 3.8+
- Jupyter Notebook
- numpy
- pandas
- matplotlib
- seaborn
- scikit-learn
