# Salary Prediction using Linear Regression

## Overview

This project focuses on predicting employee salaries based on years of experience using **Simple Linear Regression**. The analysis demonstrates a strong positive relationship between experience and salary, where salary increases consistently as experience grows.

The project includes:

- Data preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Model training
- Prediction and evaluation

---

## Dataset Information

The dataset contains:

- **YearsExperience** → Number of years of work experience
- **Salary** → Employee salary

A larger synthetic dataset of **12,000+ rows** was generated to improve training stability and simulate real-world scenarios.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Project Workflow

### 1. Data Collection

- Imported salary dataset
- Checked dataset structure and data types

### 2. Data Cleaning

- Handled missing values
- Checked duplicates
- Verified data consistency

### 3. Exploratory Data Analysis (EDA)

- Visualized salary distribution
- Analyzed relationship between experience and salary
- Detected trends and patterns

### 4. Model Building

Implemented:

- Simple Linear Regression

### 5. Model Evaluation

Evaluated model performance using:

- R² Score
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)

---

## Key Conclusion

Salary is directly proportional to years of experience, meaning as experience increases, salary also tends to increase consistently.

The dataset demonstrates a strong positive linear relationship between experience and salary, making it suitable for regression-based predictive modeling.
