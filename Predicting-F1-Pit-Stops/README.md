
# Predicting F1 Pit Stops
## Kaggle Playground Series - Season 6 Episode 5

## Project Overview
This project focuses on predicting whether a Formula 1 driver will make a pit stop on the next lap using Machine Learning techniques.

The project follows a complete Machine Learning workflow including:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Feature Scaling
- Logistic Regression Modeling
- Model Evaluation
- Kaggle Submission Generation

---

## Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn

---

## Machine Learning Workflow
1. Import Libraries  
2. Load Dataset  
3. Perform EDA  
4. Handle Categorical Features  
5. Apply Feature Scaling  
6. Train Logistic Regression Model  
7. Evaluate Model Performance  
8. Generate Kaggle Submission File  

---

## Model Used
- Logistic Regression

---

## Project Highlights
- Clean notebook structure
- Beginner-friendly workflow
- Proper preprocessing techniques
